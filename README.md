# chime-elixir
A [Chime][chime] extension for Elixir.

## Overview

This Chime extension supports the [Elixir][elixir] programming language. It integrates [ElixirLS][elixir-ls] with [ChimeKit][chimekit]'s `ExtensionProtocol`.
